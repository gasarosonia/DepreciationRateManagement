A modern, full-stack solution for tracking physical assets, managing categories, and calculating real-time financial depreciation. Built with a focus on speed, type safety, and clean UI.

Features
Real-time Depreciation: Automatically calculates "Months Owned" and "Current Book Value" using monthly straight-line depreciation logic.

Asset Lifecycle Management: Full CRUD (Create, Read, Update, Delete) for assets including purchase costs and unique Tag ID generation.

Dynamic Categories: Manage asset classes with custom monthly depreciation rates.

Modern UI: Responsive dashboard built with Tailwind CSS and shadcn/ui components.

Robust Backend: Powered by NestJS with strict DTO validation and a relational database.

Tech Stack
Frontend:

React 18 (Vite)

Tailwind CSS (Styling)

Lucide React (Icons)

Axios (API Requests)

shadcn/ui (Component Library)

Backend:

NestJS (Node.js Framework)

TypeORM (Database Mapping)

PostgreSQL/MySQL (Database)

Class-Validator (Data Integrity)
